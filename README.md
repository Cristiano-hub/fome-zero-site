# 🌱 Fome Zero e Agricultura Sustentável

Este projeto foi desenvolvido como parte das atividades extensionistas da UNINTER, com o objetivo de promover a conscientização sobre agricultura sustentável e combate à fome.

## 🧩 Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  

As imagens estão organizadas na pasta `assets`.

## 💾 Códigos-fonte
Os arquivos do projeto podem ser acessados no repositório:  
[https://github.com/Cristiano-hub/fome-zero-site](https://github.com/Cristiano-hub/fome-zero-site)

## 🎯 Objetivo
O projeto visa incentivar práticas sustentáveis, como o cultivo de hortas comunitárias e o reaproveitamento de resíduos orgânicos, promovendo alimentação saudável e responsabilidade social.
